## Aufgabe 1

Welche natürliche Zahl wird durch die Binärzahl

    10001101

dargestellt? Berechnen Sie das Ergebnis ohne Taschenrechner!

**Lösungsvorschlag**

    z = (10001101)_10
      = 1*2^0 + 0*2^1 + 1*2^2 + 1*2^3 + 0*2^4 + 0*2^5 + 0*2^6 + 1*2^7
      = 1 + 0 + 4 + 8 + 0 + 0 + 0 + 128
      = 13+128
      = 141
