## Aufgabe 11

Schreiben Sie eine Funktion, die $n!$ für $n \in \mathbb{N_0}$ berechnet (vgl. Aufgabe 8). Verwenden Sie dazu
eine for-Schleife und keine Rekursion.

Lösungsvorschlag:

    function fakultaet(int n){
      int i,n;
      for(i=1;i<=n;i++){
        f=f*i;
      }
      return f;
    }
